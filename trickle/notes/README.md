# مولد سيرفرات IPTV

## نظرة عامة
تطبيق ويب لتوليد روابط سيرفرات IPTV مجانية بشكل عشوائي.

## المميزات
- توليد روابط M3U عشوائية
- عرض معلومات السيرفر (عدد القنوات، الجودة، تاريخ الانتهاء)
- نسخ الروابط بسهولة
- عداد زمني 30 ثانية بين كل توليد
- تصميم عربي بالكامل مع دعم RTL

## البنية
- `index.html` - الصفحة الرئيسية
- `app.js` - المكون الرئيسي
- `components/` - مكونات React
- `utils/` - وظائف مساعدة

## التقنيات المستخدمة
- React 18
- TailwindCSS
- Lucide Icons
